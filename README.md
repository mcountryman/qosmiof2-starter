# quosmiof2

## Quick Start

```bash
# Install dependencies (do not use npm in this project)
yarn

# Watch for changes in API project and runs `src/main.ts` using @swc-node/register.
#   * Runs on port 3001
#   * Creates a sqlite db at <projectDir>/data/dev.db
yarn workspace @quosmiof2/api dev

# Watch for changes in web project.
#   * Runs on port 3000
yarn workspace @quisiof2/web dev
```
